# CodingWithRaviSir - C++ Assignments & Lecture Code with Java

This repository contains all my C++ assignments and lecture code from **CodingWithRaviSir**.  
The structure is simple so that it’s easy to follow and update regularly.

## 📁 Folder Structure

- **Assignments** – Contains solved problems.
  - **Assignment01.cpp**: Number triangle pattern (print 1 1 1... to i numbers per row).
  - **Assignment02.cpp**: Star/Number Patterns (33 patterns: solid triangle, inverted, hollow diamond, numbered pyramid, butterfly, etc.).
  - **Assignment03.cpp**: Binary to decimal, series 3n+2 not multiples of N2, LCM/HCF of two numbers, Nth Fibonacci, F/C conversions, prime check, GCD.
  - **Assignment04.cpp**: Array inverse, 3-sum for target, find target index, pair sum indices, max value, transport costs optimization, shopping divisors query, array reverse, square&sort array, sort 0s/1s, cyclic pair sum, bubble/selection/insertion sort, sort colors (0/1/2).

- **CodingWithRaviSir** – Contains lecture codes organized by lecture number.
  - lect01/ to lect16/ (detailed topics below)

## 📚 Lecture Details

- **lect01**: Introduction to DSA logic.
- **lect02**: Basic arithmetic (add numbers, simple interest), even/odd check, max of three, sum of natural numbers, even numbers, table of seven, reverse number.
- **lect03**: Loops for printing differences/series, coordinate straight line check, basic star patterns.
- **lect04**: Nested loops and star patterns (solid, triangle, etc.).
- **lect05**: While-loop based advanced patterns (pyramids, hollow).
- **lect06**: More patterns (hollow triangle, diamond with numbers).
- **lect07**: Prime numbers up to N, Fibonacci series, HCF/GCD, reverse number, sum of digits.
- **lect08**: Functions/methods (add, interest), conditionals (prize eligibility, coordinates), loops.
- **lect09**: Arrays - find target/max/min, swap variables, reverse array.
- **lect10**: Array rotation (LeetCode 189 - rotate by K steps).
- **lect11**: Sorting algorithms (Bubble Sort, Selection Sort, Insertion Sort).
- **lect12**: Array product except self (LeetCode 238).
- **lect13**: Maximum subarray sum (Kadane's), array duplicates.
- **lect14**: Array prefix sums, square elements & sort, two-sum for target.
- **lect15**: Binary Search for target.
- **lect16**: Advanced topics (placeholder - basic structure).

## 📌 How to Use

1. Open any `.cpp` file in your preferred C++ IDE or editor.
2. Compile and run the code to see the output.
3. Each lecture folder contains code examples taught in that session.

## 📝 Daily Updates

To update the README daily:

1. Add a new lecture folder in `CodingWithRaviSir` (e.g., `lect17`).
2. Add its details to the **Lecture Details** section above (e.g., - **lect17**: [Topic/Short Description]).
